# Day 1 - Excel Keyboard Shortcuts for Data Analysts

## Overview

Keyboard shortcuts are one of the easiest ways to improve productivity in Microsoft Excel. Instead of navigating through menus, analysts can quickly clean data, format worksheets, create tables, and analyze datasets using shortcut keys.

This guide organizes the most commonly used shortcuts into categories for everyday data analysis tasks.

# 1. Navigation Shortcuts

| Shortcut | Description |
|-----------|-------------|
| Ctrl + Arrow Key | Jump to the edge of a data region |
| Ctrl + Home | Go to cell A1 |
| Ctrl + End | Go to the last used cell |
| Home | Move to the beginning of the row |
| Ctrl + Page Up | Move to the previous worksheet |
| Ctrl + Page Down | Move to the next worksheet |
| F5 | Go To a specific cell |
| Ctrl + G | Open the Go To dialog |

---

# 2. Selection Shortcuts

| Shortcut | Description |
|-----------|-------------|
| Ctrl + A | Select all data |
| Ctrl + Shift + Arrow | Select data until the last populated cell |
| Shift + Space | Select the entire row |
| Ctrl + Space | Select the entire column |
| Ctrl + Shift + End | Select from current cell to last used cell |
| Ctrl + Shift + Home | Select from current cell to A1 |
