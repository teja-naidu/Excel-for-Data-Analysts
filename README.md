# Excel for Data Analysts

A comprehensive collection of Excel concepts, techniques, and best practices designed for aspiring and experienced Data Analysts. This repository covers everything from essential keyboard shortcuts to building interactive dashboards, providing practical knowledge that can be applied to real-world business scenarios.

---

## Repository Overview

This repository is organized into five major topics that cover the most important Excel skills required for data analysis.

### 1. Keyboard Shortcuts

Learn the most useful Excel shortcuts to improve productivity and work more efficiently.

Topics Covered:

- Navigation Shortcuts
- Selection Shortcuts
- Editing Shortcuts
- Formatting Shortcuts
- Row & Column Operations
- Formula Shortcuts
- Workbook Shortcuts
- Best Practices

---

### 2. Formulas and Functions

Understand essential Excel formulas used for data analysis and reporting.

Topics Covered:

- Mathematical Functions
- Statistical Functions
- Logical Functions
- Lookup Functions
- Text Functions
- Date & Time Functions
- Dynamic Array Functions
- Financial Functions
- Formula Best Practices

---

### 3. Data Cleaning

Learn techniques for preparing raw datasets before analysis.

Topics Covered:

- Remove Duplicates
- Text to Columns
- Flash Fill
- Find and Replace
- Data Validation
- Conditional Formatting
- Remove Blank Rows
- Convert Text to Numbers
- TRIM and CLEAN Functions
- Error Handling

---

### 4. Pivot Tables and Charts

Master Excel's most powerful analytical features for summarizing and visualizing data.

Topics Covered:

- Pivot Tables
- Pivot Charts
- Slicers
- Timelines
- Grouping Data
- Sorting and Filtering
- Calculated Fields
- Interactive Reporting

---

### 5. Dashboards

Build professional dashboards that communicate business insights effectively.

Topics Covered:

- Dashboard Fundamentals
- KPI Design
- Dashboard Components
- Interactive Dashboards
- Dashboard Design Principles
- Best Practices
- Real-World Dashboard Examples
